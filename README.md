# MystriaChronicle

### 简介

“灵境编年史²”开源内容存储库。包含但不仅限于背景故事所涉及的小说、文献、图册、人物形象、场景、世界观和故事逻辑。

### 储存库体系结构

本储存库内容包含：  
“灵境编年史²”两部作品的游戏结构和系统设计、剧情、角色设计、世界设定；  
前传“秘灵夜歌三部曲”剧情设计大纲；  
系列设定小说《古界传说：天命》剧情设计大纲；  
“灵境编年史”项目运营案（包括部分运营计划和宣传文稿），项目运营导流计划和众筹设计案；  
及部分制作过程中使用的工具和特殊方法；
